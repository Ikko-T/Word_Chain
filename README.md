Ruby対話型国名しりとりゲーム
====

## Description
このゲームの対象は4才以上です。

しりとり名人シーサと国名しりとり対決します。

しりとりゲームを通して、今まで聞いたことのない国名に出会いそして興味を持ち行ってみたいと思えたら嬉しいです。

動作例。

![Screenshot](DEMO.gif)

## Usage
下記コマンド実行でゲームが開始します。

`$ ruby game.rb`

## Install
`pry`を導入します。

`$ gem install pry`

クローンします。

`$ git clone https://github.com/Ikko-T/Word_Chain.git`
